### Example 1: Network Reconnaissance
```
1. Launch Kali GPT Advanced
2. Switch to Reconnaissance mode (Option 5)
3. Ask: "How do I perform a full reconnaissance on 192.168.1.0/24?"
4. Execute suggested nmap commands
5. Analyze results with AI
6. Get recommendations for next steps
```

### Example 2: Generate Reverse Shell
```
1. Launch Kali GPT Advanced
2. Select Payload Generator (Option 6)
3. Enter:
   - Payload type: reverse shell
   - LHOST: 10.10.14.5
   - LPORT: 4444
4. Receive multiple payload formats
5. Copy desired payload
6. Set up listener as instructed
```

### Example 3: Build Testing Workflow
```
1. Launch Kali GPT Advanced
2. Select Workflow Builder (Option 4)
3. Describe: "Full web application security test on https://target.com"
4. Get automated multi-step workflow
5. Execute each step with guidance
6. AI analyzes results at each stage
```
